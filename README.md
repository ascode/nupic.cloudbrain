# Description

CloudBrain HTM Classifier.

# Installation

	cd path/to/nupic.cloudbrain
	python setup.py develop

Or (doesn't require sudo but does require manual PYTHONPATH setup):

	python setup.py develop --prefix=/some/other/path/
